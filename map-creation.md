# Map Creation

This page is a rough outline for building the playable map itself.

## What belongs here

- Terrain and layout
- Spawn areas and routes
- Cover, sightlines, and traversal
- Objective spaces and play zones
- Props, set dressing, and readability

## Creator workflow

1. Start with a simple blockout.
2. Define the main routes and combat spaces.
3. Place objectives, spawns, and key landmarks.
4. Add props and visual detail.
5. Test how the map reads in play.
